Git 概述 ​

Git 是一个免费的开源分布式版本控制系统，旨在快速高效地处理从小到大的项目。使用 Git 可以在大量场景下高效协作。

  • GitHub flow
  • GitLab flow
  • One Flow

Git 推荐配置 ​

shell
# 阻止自动转换 CR LF
git config --global core.autocrlf false

# 阻止忽略大小写
git config --global core.ignorecase false

# 指定默认分支名为 main
git config --global init.defaultBranch main
